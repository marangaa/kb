# Fasihi Clients & Partnerships

Fasihi partners with high-growth startups and select enterprise clients to design and deploy specialized software systems.

## Safaricom (Telecommunications Partner)
Fasihi partners with Safaricom to support integration with mobile finance rails in East Africa. Our personal finance application, FinnPesa, leverages Safaricom's open APIs for secure billing transactions. Led by Sarah.

## Nairobi County (Government Client)
Fasihi was contracted by Nairobi County to design a centralized municipal feedback portal. The portal aggregates citizen requests using our Qualra adaptive feedback engine. Led by Richard.

## Kenya Airways (Aviation Partner)
Fasihi designed an automated voice support assistant for Kenya Airways, powered by Voiant. The assistant helps passengers change reservations and check flight status via natural voice calls. Led by Sarah.
