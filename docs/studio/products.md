# Fasihi Products

Fasihi builds and maintains a portfolio of specialized, AI-native products targeting analytics, customer research, and financial services.

## Voiant (Voice AI Product)
Voiant is a voice-agent software suite designed to automate call center support and operations. It features sub-100ms response latencies and can handle complex, multi-turn phone support interactions naturally.

## Resonate (Product Analytics)
Resonate is a product analytics platform that aggregates user activity and feedback data, converting qualitative comments into actionable product insights. It groups customer feature requests automatically using vector embeddings.

## Qualra (Customer Research)
Qualra is an AI-driven research platform that conducts adaptive interviews. Unlike static survey tools, Qualra adapts its questions dynamically based on the participant's previous answers to uncover core user needs.

## Finn (FinnPesa)
Finn (FinnPesa) is a personal financial management application optimized for the Kenyan market. It integrates with mobile payment APIs to help users track transactions, budget spending, and hit savings goals.
