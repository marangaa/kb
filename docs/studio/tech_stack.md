# Fasihi Core Technology Stack

Fasihi relies on a modern, open-source-first engineering stack optimized for low-latency AI inference and robust web experiences.

## Frontend Technologies
* **Next.js & React:** Our primary framework for rendering fast, SEO-friendly web interfaces and interactive components. Led by James.
* **TypeScript:** Ensures static typing safety across all client applications.

## Backend & Databases
* **FastAPI & Python:** Used for running our core AI engines and orchestration endpoints.
* **AWS Aurora PostgreSQL (with pgvector):** Stores our relational datasets and handles high-performance semantic vector similarity searches. Led by Sarah.

## AI & Frameworks
* **EVE Agent Framework:** Used to configure durable, session-aware agent runtimes. Led by Amina.
* **LightRAG:** Powers our incremental GraphRAG pipeline for self-structuring organizational memory. Led by Amina.
