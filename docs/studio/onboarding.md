# Fasihi New Hire Onboarding Guide

Welcome to Fasihi! This document provides information to help you navigate your first week as a software engineer or designer at our Nairobi studio.

## Hardware & Environment Setup
* **Machine Provisioning:** All engineers receive an Apple MacBook Pro (M3 Pro, 16-inch). Design team members receive a calibrated studio display. James manages hardware requests.
* **VPN & Network Access:** We enforce zero-trust network access (ZTNA) via Cloudflare Access. Sarah handles user group permissions.

## Communication Channels
* **Slack:** Our primary real-time communication platform.
* **Notion:** Used for documentation, design specs, and meeting notes.
* **Linear:** Used strictly for tracking sprint tasks, bugs, and product releases.

## First Week Tasks
1. Complete your profile setup on Slack and GitHub.
2. Clone the repository and run the local development server.
3. Schedule 15-minute syncs with Amina (to understand EVE core agents) and Sarah (to set up AWS database profiles).
